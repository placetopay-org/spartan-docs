Color system

Unique, accessible and consistent.

The color system helps you reflect our brand and style creating a visual pattern in every component aspect. We select a harmonious cool gray color palette and secondary colors to provide accessibility, simplicity and create a consistent experience in all our products ensuring color contrast between the elements.

Primary palette

Is focused in cool gray tones with varying degrees of saturation to allow dark and light variants to applied in UI in different ways as backgrounds or text
